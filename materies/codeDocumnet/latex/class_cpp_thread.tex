\doxysubsection{Detailed Description}
GNU GENERAL PUBLIC LICENSE Version 3, 29 June 2007

(C) 2020-\/2022, Bernd Porr \href{mailto:mail@bernporr.me.uk}{\texttt{ mail@bernporr.\+me.\+uk}} A thin wrapper around the C++ thread model to avoid a static callback. Instead just inherit this class and overload \mbox{\hyperlink{class_cpp_thread_a792b79e72250710147c452648def4a78}{run()}} which then runs in this thread. This is header-\/only so that it can be performed inline for max performance.
